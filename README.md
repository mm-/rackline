# Rackline

Gem for interacting with the Rackspace Cloud via command line

## Installation

Add this line to your application's Gemfile:

    gem 'rackline'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rackline

## Usage

rackline --help
